# House Price Prediction using Linear Regression

## Overview
This project predicts house prices using Linear Regression based on:
- Living Area
- Number of Bedrooms
- Number of Bathrooms

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Machine Learning Algorithm
- Linear Regression

## Dataset
House Prices - Advanced Regression Techniques Dataset from Kaggle

## Output
The model predicts house prices and visualizes Actual vs Predicted Prices using a scatter plot.
